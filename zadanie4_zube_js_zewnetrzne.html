Odpowiedzi na pytania o JavaScript

  1. Kiedy powstał JavaScript?
    JavaScript powstał 4 grudnia 1995 roku.
  2. Dlaczego jest to język skryptowy?
    Jest to język skryptowy, ponieważ działa w środowisku interpretera i jest używany do automatyzacji zadań w aplikacjach (np. przeglądarkach).
  3. Gdzie jest wykonywany JS (klient lub serwer)?
    JavaScript jest wykonywany zarówno po stronie klienta (przeglądarka), jak i serwera (np. Node.js).
  4. Dlaczego JS to bezpieczne narzędzie?
    JS jest bezpieczny, ponieważ działa w tzw. sandboxie, co ogranicza jego dostęp do systemu operacyjnego użytkownika.
  5. Czy jest to język obiektowy, jeśli tak to dlaczego?
    Tak, JavaScript jest językiem obiektowym, ponieważ wspiera programowanie obiektowe za pomocą prototypów i klas.
  6. Zalety JS:
    • Zaleta 1: Jest uniwersalny – działa w przeglądarkach i na serwerze.
    • Zaleta 2: Dynamicznie aktualizuje zawartość stron internetowych.
    • Zaleta 3: Wspiera szeroki ekosystem bibliotek i frameworków.
  7. Framework to:
    Framework to zestaw narzędzi i bibliotek, które ułatwiają tworzenie aplikacji, zapewniając gotowe komponenty i struktury.
  8. Do aplikacji mobilnych można użyć Framework?
    Tak, np. React Native lub Ionic.
  9. Aplikacje responsywne to:
    Aplikacje dostosowujące się automatycznie do różnych rozdzielczości ekranów (np. desktop, tablet, telefon).
  10. Jak działa ten fragment kodu?
    <input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">
    Tworzy przycisk o nazwie „Nowa Strona”. Po jego kliknięciu wywoływana jest funkcja JavaScript „WinOpen(' ')”, która prawdopodobnie otwiera nowe okno lub wykonuje inną akcję.
